# Real-vs.-Synthetic-Data-Performance-Across-NLP.,

The main objective of this study is to examine the extent to which real and synthetic texts affect the success of different algorithmic architectures. In this study, a synthetic copy of each row in the original dataset was created with different sentences and a completely different scenario, preserving the semantic features of each row. The original data was taken from comments by real users on a movie review site and has a binary classification of positive and negative tags. The generated synthetic data was constructed with the same tags but a completely different movie topic. The publicly available LLM model “llama-3.1-8b-instant” was used for generation according to these criteria. To measure the response of different algorithmic approaches to the data, linear algorithms (Naive Bayes, Logistic Regression, Support Vector Machines), tree-based ensemble algorithms (Random Forest, XGBoost), sequential network model (LSTM), and advanced language architecture (RoBERTa) were used.
